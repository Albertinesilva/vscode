Referência Rápida de CSS com Emmet

1. Posicionamento

pos-r /* position: relative; */ pos-a /* position: absolute; */ pos-f /* position: fixed; */ pos-s /* position: sticky; */

2. Margens (margin)

m10 /* margin: 10px; */ m10-20 /* margin: 10px 20px; */ m-a /* margin: auto; */

3. Espaçamento Interno (padding)

p10 /* padding: 10px; */ p10-20 /* padding: 10px 20px; */

4. Tamanhos (width e height)

w100 /* width: 100px; */ h50p /* height: 50%; */ w100vw /* width: 100vw; */

5. Exibição (display)

d-b /* display: block; */ d-i /* display: inline; */ d-f /* display: flex; */

6. Float e Clear

fl-l /* float: left; */ fl-r /* float: right; */ cl-b /* clear: both; */

7. Alinhamento de Texto (text-align)

ta-c /* text-align: center; */ ta-r /* text-align: right; */

8. Flexbox

d-f /* display: flex; */ fd-c /* flex-direction: column; */ ai-c /* align-items: center; */ jc-sb /* justify-content: space-between; */

9. Grid Layout

d-g /* display: grid; */ gtc-3 /* grid-template-columns: repeat(3, 1fr); */

10. Bordas e Radius

bd1-s /* border: 1px solid; */ br10 /* border-radius: 10px; */

11. Background e Cores

bgc-r /* background-color: red; */ co-w /* color: white; */

12. Fontes e Tamanhos de Texto

fs20 /* font-size: 20px; */ fwb /* font-weight: bold; */

13. Box Shadow e Opacity

bxsh /* box-shadow: 0px 4px 10px rgba(0, 0, 0, 0.3); */ op8 /* opacity: 0.8; */
